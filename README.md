# 🚀 ASP.NET Core MVC Learning Projects – Saipriya Kaja

This repository contains a collection of mini-projects and exercises built while learning ASP.NET Core MVC and .NET Full Stack Development. Each folder represents a separate application or concept designed to reinforce both backend and frontend development fundamentals using .NET and related technologies.
These projects explore core features such as ASP.NET Core MVC, Web API, Entity Framework Core, Razor Views, SQL Server, Authentication & Authorization, and integration with front-end frameworks like JavaScript or HTML/CSS.

## 🧠 Topics Covered

- ✅ ASP.NET Core MVC project structure and lifecycle
- 🔁 RESTful API development using ASP.NET Core Web API
- 🗃️ CRUD operations with Entity Framework Core and SQL Server
- 🔐 Authentication & Authorization with ASP.NET Identity
- 🧪 Unit testing with xUnit
- 🛠️ Connecting to relational databases with EF Core
- ⚙️ Application configuration using appsettings.json
- 🌐 Serving web pages with Razor Views

## ✅ How to Run
1. Clone the repository:
git clone https://github.com/saipriyak2/dotnetcore-mvc-projects.git
cd dotnetcore-mvc-projects
2. Open the desired project  in Visual Studio 2022 or VS Code.
3. Run the project:
In Visual Studio, press F5 (Run with Debug) or Ctrl+F5 (Run without Debug)
Or using .NET CLI from project folder:

dotnet build
dotnet run
dotnetcore-mvc-projects

By default, the application will run on:
https://localhost:5001  (HTTPS)
http://localhost:5000   (HTTP)
(Port or path may vary depending on the project)

## 🔐 Note About Secrets
